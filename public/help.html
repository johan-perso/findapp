Quésaco ?

FindApp vous permet de trouver l'application que vous recherchez parmis plusieurs sources, que la communeauté peut créer. FindApp offre également 2 sources actives par défaut (avec environ 300 PWAs et 42400 applications natives).

Comment ça marche ?

Lorsque vous faites une recherche, l'ensemble des applications de toutes les sources que vous avez ajoutés seront analysés afin de trouver les applications que vous souhaiter.
Pour obtenir des informations sur comment marche une source, rendez-vous sur la page de gestion des sources.

Recherche avancé

Il est possible d'utiliser certains termes de recherches afin de rendre les résultats plus précis.

Element Description
=name Application dont le nom est exacte à name
^name Applications qui commencent par name
!^name Applications qui ne commencent pas par name
'name Applications qui contiennent name
!name Applications qui ne contiennent pas name
name$ Applications qui finissent par name
!name$ Applications qui ne finissent pas par name
_all Toutes les applications, ne prend pas en compte le terme à rechercher
_random Toutes les applications, dans un ordre aléatoire
_noIcons Toutes les applications n'ayant pas d'icônes
_pwa Toutes les PWAs (applications web)
